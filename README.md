# 게임(B-6조)
C# 언어와 유니티 엔진을 활용하여 만든 2.5D 공포 탐험 게임입니다.

## 👨‍🏫 프로젝트 소개
내일배움캠프에 참여하고 계신 팀원 권용님의 아이디어로 탄생한 우화를 기반으로하는 게임입니다.
